Detailed Description

This structure is a user-friendly overlay for the display control registers.

You may configure the overlay by setting the bit fields declared in the base class, custard::display::_dispcnt_overlay. In addition, this structure provides a collection of methods for setting configurations compatible with both the main (A) and sub (B) 2D rendering engines.

Methods set_main and set_sub write the overlay to the main (A) engine and sub (B) engine display control registers, respectively.

Methods of the form graphics_XXXX, where each X is one of T, A or E, configure the overlay for 2D graphics display with background modes as follows:

XXXX BG0 BG1 BG2 BG3
TTTT Text Text Text Text
TTTA Text Text Text Affine
TTAA Text Text Affine Affine
TTTE Text Text Extended Extended
TTAE Text Text Affine Extended
TTEE Text Text Extended Extended
