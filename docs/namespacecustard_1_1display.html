custard::display Namespace Reference

Detailed Description

Display control overlays.

This namespace contains functions and classes for configuring the main (A) and sub (B) 2D rendering engines. Structures config and main_config store display control register overlays, allowing to configure the rendering engines in an object-oriented way. Inline functions are provided for immediately setting well known configurations.

See also
GBATEK's documentation for NDS display control: http://problemkaputt.de/gbatek.htm#dsvideobgmodescontrol
